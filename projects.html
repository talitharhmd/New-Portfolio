Research Project 2024

Automated Essay Scoring System for Indonesian Text

Developing and comparing AI models to automatically grade student essays by measuring semantic similarity to reference answers.

Goal: To create a faster, more objective, and automated way to grade student reflections using AI.

Key & Responsibilities:

  • Preprocessed raw text data from student surveys by removing stop words and applying stemming using Sastrawi.
  • Built and compared text-processing models (Word2Vec vs. FastText) to evaluate semantic understanding performance.
  • Proved the FastText model's superior accuracy (0.80 score) compared to Word2Vec (0.53) for Indonesian text processing.
Python | Gensim (FastText) | NLTK | Sastrawi
Methods: Natural Language Processing (NLP), Text Preprocessing, Cosine Similarity
Github Repository
